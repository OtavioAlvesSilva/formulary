# 📧 Contact Form

> A clean and responsive contact form built with HTML and CSS.

🌐 **Live Demo:** [View Project](https://otavioalvessilva.github.io/contact-form)

---

## 📖 About the Project

This is a **contact form** developed as a study project for **HTML and CSS**. The form features a modern and clean design, with fields for name, email, phone, subject, priority, message, and a consent checkbox.

The project demonstrates:
- Semantic HTML structure
- Advanced CSS styling
- Responsive design for all devices
- Form validation with `required` attributes
- Custom styled radio buttons and checkbox

---

## ✨ Features

- ✅ Fully responsive design (desktop, tablet, mobile)
- ✅ Form validation with `required` fields
- ✅ Custom styled radio buttons (Priority: Low, Medium, High)
- ✅ Custom styled checkbox (Consent)
- ✅ Clean and modern UI
- ✅ Accessible form elements
- ✅ Footer with credits and GitHub link

## 📋 Form Fields

| Field | Type | Required |
|---|---|---|
| Name | Text input | ✅ Yes |
| Email | Email input | ✅ Yes |
| Phone | Tel input | ✅ Yes |
| Subject | Select dropdown | ✅ Yes |
| Priority | Radio buttons | No |
| Message | Textarea | No |
| Consent | Checkbox | No |
